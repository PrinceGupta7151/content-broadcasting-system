# Content Broadcasting System - Frontend

A modern React-based educational content broadcasting system where teachers upload subject-based content, principals approve/reject it, and students view live broadcasts.

## Features

- **Authentication**: Secure login with role-based access
- **Teacher Dashboard**: Upload content, track status, view submissions
- **Principal Dashboard**: Review, approve, and reject content with feedback
- **Public Broadcast**: View live content without authentication
- **Responsive Design**: Works on desktop, tablet, and mobile
- **Real-time Feedback**: Toast notifications for all user actions
- **File Validation**: Type and size validation for uploads
- **Scheduling**: Schedule content for specific time windows

## Tech Stack

- **Frontend**: React 18.2
- **Routing**: React Router 6
- **Forms**: React Hook Form + Zod validation
- **HTTP Client**: Axios with interceptors
- **Styling**: Tailwind CSS
- **State Management**: React Context API
- **Icons**: Lucide React
